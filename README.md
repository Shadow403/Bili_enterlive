# Bili_enterlive
- 环境版本
    - 1.php - 7.4.5
    - 2.Apche - 24
- 使用方法
    - 1.将文件整体放入htdocs/你的域名文件夹下
    - 2.确保php已开启gb和cURL
    - 3.开启Apache服务

```
localhost/enterlive/?uid=<int> //<- 查看生成图片端口
```
